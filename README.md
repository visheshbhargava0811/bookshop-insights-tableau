# bookshop-insights-tableau
A stakeholder-driven analytics project built on the Bookshop dataset — 9 related tables totaling ~107K rows (56K sales transactions, 50K reviews) modelled in Tableau with 28 calculated fields. The analysis is organized into three stories — Customer Success, Customer Experience, and Inventory Management — each containing 3 dashboards scoped to that team's requirements. Net revenue analysis joining sales to edition/book data surfaced that SciFi/Fantasy drove 42% of Q1's $695K in revenue, more than double Mystery's 15%, feeding an inventory dashboard that flags understocked genres.
